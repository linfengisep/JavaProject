1.Generic type
1.1 format: class name <T,T1,T2..>{}
T1,T2, : type parameter
1.2 type naming convention
E - Element (used extensively by the Java Collections Framework)
K - Key
N - Number
T - Type
V - Value
S,U,V etc. - 2nd, 3rd, 4th types
1.3 Invoking and Instantiating a Generic Type:
replaces T with some concrete value, such as Integer
